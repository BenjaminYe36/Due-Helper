## 测试与编译
### 测试所需环境

- NodeJs & npm

### 测试步骤指南
1. 用git克隆到本地 或 下载源码压缩包并解压
2. 在项目文件夹下运行 `npm install`
3. 运行 `npm run dev` 来启动测试

### 编译，打包
运行命令 `npm run build` 即可
